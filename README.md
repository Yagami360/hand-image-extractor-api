# hand-image-extractor
人物パースモデル（Graphonomy）と DensePose を用いて、指定した人物画像から手領域の画像を抽出します。

## ■ 動作環境
- NVIDIA Tesla K80
- docker
- docker-compose
- nvidia-docker 2.0

## ■ 使用法

```sh
$ sh run_api.sh
```
